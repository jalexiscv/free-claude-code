⚙️ Configuration

📝 .env Example

# DeepSeek (recommended)
DEEPSEEK_API_KEY="sk-..."
MODEL="deepseek/deepseek-v4-flash"

# Or NVIDIA NIM
NVIDIA_NIM_API_KEY="nvapi-..."
MODEL="nvidia_nim/nvidia/nemotron-3-super-120b-a12b"

ANTHROPIC_AUTH_TOKEN="freecc"

▶️ Run Server

# Start the proxy
uv run uvicorn server:app
--host 0.0.0.0 --port 8082

Then connect Claude Code:

claude --proxy-url http://localhost:8082
--api-key freecc

🚀 Quick Start

1️⃣ Clone & Install

git clone https://github.com/jalexiscv/free-claude-code.git
cd free-claude-code
uv sync

2️⃣ Configure

Copy .env.example to .env, edit your API key and model:

cp .env.example .env
nano .env

3️⃣ Run

uv run uvicorn server:app --host 0.0.0.0 --port 8082

4️⃣ Connect Claude Code

claude --proxy-url http://localhost:8082 --api-key freecc
